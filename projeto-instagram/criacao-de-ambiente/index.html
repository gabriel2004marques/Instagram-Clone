Vamos criar um projeto em React do zero.

Para aprender e testar o React, você deve configurar um Ambiente React em seu computador.

A ferramenta create-react-app é uma maneira oficialmente suportada de criar aplicativos React.

O Node.js é necessário para usar o app create-react.
Você pode baixá-lo aqui.
Node.JS é uma ferramenta feita para criar programas utilizando Javascript

Abra seu terminal na pasta em que deseja criar seu aplicativo. Em aula utilizamos um diretório padrão para colocar nossos projetos em React. A pasta em si não importa, mas saber onde está o diretório é essencial.

Execute este comando para criar um aplicativo React chamado nome-do-projeto:

npx create-react-app nome-do-projeto

O comando acima configurará tudo o que você precisa para executar um aplicativo React. Esse processo pode levar alguns minutos Uma vez que a mensagem "Happy Hacking" apareça, o seu projeto esta pronto para ser executado.

Rodando nossa aplicação

Execute este comando para navegar para o diretório nome-do-projeto:

cd nome-do-projeto

E então execute o projeto:

npm start

Nesse ponto, caso seja a sua primeira vez executando um app em React você será perguntado se deseja proceder. Diga que sim ("y"). Uma nova janela do seu navegador padrão irá aparecer com seu recém-criado React App! Caso contrário, abra seu navegador e digite localhost:3000 na barra de endereços.

E se não for criar do zero?

Caso você baixe um template ou um projeto e queira executá-lo você pode navegar para a pasta onde esta seu projeto

cd "C:\Users\Public\nome-do-projeto"

E então executar a instalação que consiste em baixar a pasta node_modules e todas as suas dependências.

npm install

E então executar o projeto

npm start
